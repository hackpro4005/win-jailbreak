# win-jailbreak

Jailbreak windows

Welcome, this scripts are for jailbreaking Windows follow the steps bellow
WARNING: RECKLESS EDITING OF SYSTEM FILES CAN BRICK YOUR WINDOWS INSTALL



Step 1: Find and note the SID of TrustedInstaller

Step 2: Flash a bootable iso to a usb drive

Step 3: Make a new profile called "SUPERUSER"

Step 4: Backup your SAM

Step 5: Shut down windows and boot to your usb

Step 6: Edit the SAM so SUPERUSER has the SID of TrustedInstaller

Step 7: Shut down and boot back into Windows

Step 8: Run the installer as admin
